# JavaIM Docs

这里是JavaIM的文档！

{% hint style="info" %}
**注意：** JavaIM仍然处于开发阶段，不能保证通信的安全性。
{% endhint %}

## 概述

> JavaIM是一款使用Java编写的跨平台加密通信软件。
>
> JavaIM目前通过RSA算法对通信内容进行加密，这种方式仍然存在一些漏洞，将来将会推出更加安全可靠的方式进行传输。
