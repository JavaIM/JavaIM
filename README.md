## 概述

> JavaIM是一款使用Java编写的跨平台加密通信软件。
>
> JavaIM目前通过RSA+AES算法对通信内容进行加密，但是由于RSA客户端的公钥传输使用明文，这种方式仍然存在一些漏洞。
