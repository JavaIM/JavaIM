> 注意： JavaIM仍然处于开发阶段，不能保证通信的安全性。

JavaIM是一款使用Java编写的加密通信软件。

JavaIM目前通过RSA+AES算法对通信内容进行加密。

项目维护现在主要由AlexLiuDev233负责。


本仓库已经在2023年5月21日迁移至[此处](https://github.com/JavaIM/JavaIM)
